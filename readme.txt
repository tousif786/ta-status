TA Status plugin allow admin (network or sub site admin) to post status and that will appear on dashboard widget.

== Description ==

Allow admin (network or sub site admin) to post status on TA Status settings page. Admin can see that status on
dashboard widget. Network admin can overwrite status of all subsite.

Some cool features this plugin provides:

* Add status that will appear on dashboard widget.
* Network admin can overwrite all sub site status.

== Installation ==

Following are the steps to install and activate TA Status:

1. Upload `ta-status` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Is this plugin can be used on WordPress multisite installation? =

Yes, it can be installed on multisite as well as single site installation.

= Where I can see the posted status? =

Posted status will be appeared on dashboard widget.

== Screenshots ==

1. Plugin can be activated network wide from network admin plugin page.
2. Also, admin can activate this plugin on single site.
3. Plugin settings page can be found under tool TA Satus.
4. Finally, posted status will be appeared on dashboard widget.
